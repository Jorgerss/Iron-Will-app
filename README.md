# Iron-Will-app
App de disciplina extrema con IA de Google para bloqueos de 24 horas y validación de hábitos mediante cámara.
PROYECTO: IRON WILL (Voluntad de Hierro)
Busco Desarrollador para App de Disciplina Extrema
Nota inicial: No soy programador. Estoy construyendo esto por una necesidad personal de autodisciplina. Busco a alguien que domine el código para convertir este concepto en una realidad funcional. Si buscas una app "zen" o de "bienestar", este no es el lugar.
1. El Concepto: La Última Elección
IRON WILL nace bajo una premisa simple: "Si esto no funciona, es mejor no intentarlo de nuevo". Es una aplicación diseñada para cuando ya no hay más sueño en donde arrastrarse y la voluntad propia ha fallado. No es un asistente, es un ejecutor.
2. Lo que ya está hecho (Idealización en AI Studio)
He utilizado Google AI Studio para desarrollar el "cerebro" de la aplicación mediante Prompt Engineering y System Instructions.
Ya existe una lógica de respuesta fría y neutral.
Ya están configurados los parámetros de visión artificial para reconocer objetos (libros, agua, equipo de gimnasio, etc.).
La IA ya sabe cómo transformar una confesión de un mal hábito en una estructura de bloqueo lógica.
3. Funcionalidades Clave que Necesito Desarrollar:
Bloqueo de Hierro: Restricción total de aplicaciones y sitios web por ciclos de 24 horas. Sin botones de "posponer".
Alarma Industrial: Un sistema de audio que ignore el modo silencio y solo se detenga ante una "Prueba de Vida".
Validación Visual: Integración con la cámara para que la IA confirme en tiempo real (no fotos de galería) que el usuario ha cumplido su tarea.
Persistencia de Sistema: Medidas para evitar que la app sea desinstalada o forzada a detenerse durante un ciclo de bloqueo.
